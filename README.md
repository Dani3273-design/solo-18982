# 3D 迷宫滚球游戏

一款使用 Pygame 开发的 3D 迷宫滚球游戏。通过鼠标拖拽控制迷宫倾斜，让小球在重力作用下滚动，最终将小球送入出口。

## 游戏特性

- **3D 等距视角**: 俯视斜 45° 观看迷宫，呈现立体效果
- **随机迷宫生成**: 每次游戏都生成不同的迷宫
- **物理引擎**: 小球具有真实的物理特性（重力、惯性、摩擦力、碰撞反弹）
- **鼠标控制**: 拖拽鼠标控制迷宫倾斜方向
- **计时系统**: 记录通关时间
- **多线程**: 迷宫生成使用独立线程，保证游戏流畅


## 游戏说明

### 操作方法

1. **开始游戏**: 在初始页面点击 "开始游戏" 按钮
2. **控制迷宫**: 按住鼠标左键并拖拽，迷宫会随拖拽方向倾斜
3. **小球滚动**: 迷宫倾斜后，小球会受重力影响向倾斜方向滚动
4. **通关条件**: 将小球滚入黑色洞口（有橙色边框标记）
5. **重新开始**: 在结算页面点击 "重新开始" 按钮

### 游戏规则

- 每次开始游戏都会随机生成新的迷宫
- 小球的初始位置远离出口，但保证可以到达
- 迷宫有墙壁阻挡，小球碰到墙壁会反弹
- 松开鼠标后，迷宫会缓慢恢复到水平状态
- 小球具有惯性，不会立即停止移动

### 评分标准

- **3 星**: 30 秒内通关
- **2 星**: 30-60 秒内通关
- **1 星**: 60 秒以上通关

## 技术说明

### 核心模块

1. **labyrinth.py**:
   - `LabyrinthGenerator`: 使用回溯法随机生成迷宫，BFS 算法查找最远可达点作为起点和终点
   - `Labyrinth3D`: 实现等距投影（Isometric Projection），将 2D 坐标转换为 3D 视觉效果

2. **ball.py**:
   - 实现完整的物理引擎，包括重力、摩擦力、碰撞检测与反弹

3. **control.py**:
   - 处理鼠标拖拽事件，将鼠标移动转换为迷宫倾斜角度

4. **ui.py**:
   - 实现四个游戏界面：开始界面、加载界面、游戏界面、结算界面
   - 按钮组件支持悬停效果

5. **time.py**:
   - 高精度计时器，支持开始、暂停、恢复、停止操作
